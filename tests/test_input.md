# Simiki #

Simiki is a simple static site generator for wiki. Powered by Python.

* Support [Markdown](http://daringfireball.net/projects/markdown/).
* Generate static Htmls.
* Have a CLI tool.
* Simple for wiki.

## Why named "Simiki" ##

Simki is **Sim**ple W**iki** .
